# java-blackjack

## 구현 기능
-  게임에 참가할 사람 이름 입력
-  카드 나눠주기
  - 정책: 사용하는 카드에 대해 정의
    - 카드의 종류: 하트, 다이아, 클로버, 스페이드
    - 카드의 숫자: A, 2 부터 10, J, Q, K
  - 카드 생성 기능이 필요
  - 카드 뽑기 기능이 필요
    - 딜러는 자기 자신의 카드도 뽑는다.
      - 딜러는 공개카드 1장과 비공개 카드 1장씩 받고, 공개 카드 출력
    - 참여자 별 카드 주기
  - 참여자들 카드 목록 출력
-  플레이어 순서대로 카드를 더 받을 것인지 확인
  - 플레이어가 카드를 받았을 경우 현재 카드 목록 출력
- 딜러의 Bust 판정
  - 딜러의 카드 합이 16이하면 카드 1장 받기
- 딜러 카드와 플레이어 카드 출력하기
  - 카드 목록 출력
  - 카드 숫자 합 출력
- 최종 승패여부 출력
  - 딜러의 승패 출력
  - 플레이어별 승패 출력
