# java-blackjack

블랙잭 미션 저장소

# 기능 명세서

## 입력

- [x]  참여 인원 이름을 입력받는다.
    - [x]  게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)
- [x]  플레이어별로 히트/스탠드를 입력받는다.
    - [x]  pobi는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)
- [x] 플레이어별로 배팅 금액을 입력받는다.
    - [x] 배팅 금액은 음수가 아니어야 한다.

## 출력

- [x]  카드 분배 기록을 출력한다.

  ```java
  딜러와 pobi,jason에게 2장을 나누었습니다.
        딜러카드:3다이아몬드
        pobi카드:2하트,8스페이드
        jason카드:7클로버,K스페이드
  ```

- [x]  플레이어 보유 카드를 출력한다.

  ```java
  pobi카드:2하트,8스페이드,A클로버
  ```

- [x] 딜러 카드 추가 여부 출력
   ```markdown
   딜러는 16이하라 한장의 카드를 더 받았습니다.
   
   딜러는 17이상이라 카드를 받지 않았습니다.
   ```
- [x]  참가자의 결과를 최종 보유 카드와 합계를 출력한다.
  ```markdown
  딜러카드: 3다이아몬드, 9클로버, 8다이아몬드 - 결과: 20
  pobi카드: 2하트, 8스페이드, A클로버 - 결과: 21
  jason카드: 7클로버, K스페이드 - 결과: 17
  ```
- [x]  최종 승패를 출력한다.
  ```markdown
  딜러: 1승 1패
  pobi: 승 
  jason: 패
  ```
- [x] 최종 수익을 출력한다.

``` markdown
딜러: 10000
pobi: 10000 
jason: -20000
```

## 핵심 기능

- [x]  카드 덱 생성 및 셔플
- [x] 카드 점수 계산
- [x]  Ace 1/11 처리
- [x]  버스트 판정
- [x]  히트/스탠드 처리
- [x] 블랙잭이면 히트/스탠드 단계를 거치지 않고 블랙잭으로 승리한다.
- [x] 플레이어 버스트 시 더 이상 카드를 뽑을 수 없다.
- [x] 승패 판정
- [x] 플레이어와 딜러 모두 블랙잭이면 무승부 처리한다.
- [x] 딜러가 21을 초과하면 버스트가 아닌 모든 플레이어는 승리 처리한다.
- [x] 배팅 기능
    - [x] 일반 승리 시 배팅 금액의 1.0배를 받는다.
    - [x] 블랙잭 승리 시 배팅 금액의 1.5배를 받는다.
    - [x] 일반 패배 시 배팅 금액을 모두 잃는다.
    - [x] 버스트 시 경우 배팅 금액을 모두 잃는다.

## 유효성 검사

### 이름

- [x]  빈 값을 입력한 경우
- [x]  1 ~ 10자 범위를 벗어나는 경우

### 히트/스탠드

- [x]  y 또는 n이 아닌 값을 입력한 경우
