# java-blackjack

블랙잭 미션 저장소


## 구현할 기능 목록

- 게임에 참여할 인원 이름 입력받기
- 딜러, 플레이어에게 카드 2장 분배
- 플레이어 순서대로 힛/스테이 물어보기
  - 힛 선택 시, 카드 한 장을 더 받는다.
  - 스테이 선택 시, 카드를 더이상 받지 못한다. 
  - 카드 합계가 21을 초과하는 경우 버스트 판정
- 딜러 카드 합계에 따라 다음 행동을 수행한다.
  - 17 미만일 경우, 카드를 더 받는 행동을 반복한다.
  - 17 이상일 경우, 카드를 받지 않는다.
- 딜러와 플레이어의 모든 카드와 합계를 출력한다.
- 최종 결과를 출력한다.
