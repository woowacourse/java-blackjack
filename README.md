# java-blackjack

블랙잭 미션 저장소

## 입력

- [x] 게임에 참여할 사람의 이름 입력 받기(구분자 ,)
    - [x] 공백일 경우 `IllegalArgumentException`을 발생시킨다.
- [x] 플레이어로부터 베팅 금액 입력 받기
    - [x] 공백일 경우 `IllegalArgumentException`을 발생시킨다.
    - [x] 숫자 형식이 아닐 경우 `IllegalArgumentException`을 발생시킨다.
- [x] 플레이어에게 hit 여부 입력 받기(y,n).
    - [x] 공백일 경우 `IllegalArgumentException`을 발생시킨다.

## 비지니스 기능

- 플레이어 등록
    - [x] 최소 2자 ~ 최대 7자, 이외 범위는 `IllegalArgumentException`을 발생시킨다.
    - [x] 영어&한글만 가능, 이외 형식은 `IllegalArgumentException`을 발생시킨다.
    - [x] 중복된 이름이 존재할 경우 `IllegalArgumentException`을 발생시킨다.

- 게임 초기화
    - [x] 게임 시작 시, 모든 카드를 덱에 등록한다.
    - [x] 카드를 섞는다.
    - [x] 딜러와 플레이어는 카드를 받는다.

- 카드 점수 합산
    - [x] 카드 합산 점수를 계산한다.
        - [x] Ace 점수(1/11)를 처리한다.

- 게임 진행 판정
    - [x] 딜러와 플레이어의 히트/스탠드 여부를 판정한다.
    - [x] Bust 여부(21 초과)를 판정한다.

- 최종 수익 계산
    - [] 각 참여자마다 베팅으로부터 얻은 수익을 계산한다.
        - [x] 베팅 가능한 금액 범위는 `최소 5_000 ~ 최대 200_000원`이며, 이외 범위는 `IllegalArgumentException`을 발생시킨다.
        - [] 블랙잭, 버스트 여부에 따라 배팅 금액을 다르게 계산한다.

## 출력

- [x] 이름과 함꼐 게임 시작 문구 출력
- [x] 가지고 있는 카드 목록 출력
    - [x] 딜러는 맨 처음 받은 한 장은 출력 x
- [x] 카드 합산 결과 출력
- [] 딜러 및 각 플레이어별 최종 수익 결과 출력
