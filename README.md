# java-blackjack

블랙잭 미션 저장소

## 구현 기능 내용

### 입력 기능

- [x] 참여자 이름을 입력받는다.
  - 구분자를 기준으로 입력받는다.
- [x] 카드를 플레이어가 추가로 받을 건지 여부를 입력받는다.
- [ ] 베팅 금액을 입력받는다.

### 출력 기능

- [x] 카드를 참여자에게 몇 장 나눠줬는지 출력
- [x] 참여자가 받은 실제 카드를 모두 출력
  - 딜러는 1장만 출력
- [x] 플레이어가 카드를 더 받게됐을 때 현재 카드 내역을 출력
- [x] 딜러가 카드를 한장 더 받는지 여부를 출력
- [x] 최종적으로 각각의 참여자들이 갖고 있는 카드 내역과 점수 출력
- [x] 최종 수익을 출력

### 핵심 기능

- [x] 참여자가 카드를 한장 뽑는 기능
- [x] 참여자가 갖고 있는 카드를 오픈하는 기능
  - 딜러일 때 처음 오픈한다면 맨 앞 카드만 공개, 아니면 모든 카드를 공개
  - 플레이어일 때 모든 카드를 공개
- [x] 참여자에게 카드를 추가로 한 장 더 주는 기능
  - 딜러는 16 이하면 받는다.
  - 플레이어는 입력 값(y/n)에 따라 받는다.
- [x] 합계 점수 계산 기능
- [x] 딜러의 카드 패(hand) 전체 오픈 기능
- [x] 최종 승패 계산 기능
  - 플레이어와 딜러의 상태에 따라 다음 표와 같이 승패를 구분할 수 있다.
    ![블랙잭-승패-판정-표](https://github.com/user-attachments/assets/54986bf6-924d-45a2-82aa-b51b34e6ffa4)
- [x] 카드 점수 계산 기능
  - 숫자 카드는 숫자 그대로
  - J/Q/K는 10점
  - ACE는 바로 아래에서 설명한다.
- [x] 합계에 따라 Ace의 값을 1 또는 11로 계산하는 기능
  - 플레이어가 유리한 방향으로 계산한다.
  - 한번 계산한 값은 유리한 방향에 따라 변경될 수 있다.
  - 예시
    - 5하트, A클로버 -> A는 11로 계산. 합계 16.
    - (카드 한 장 추가) 5하트, A클로버, 10스페이드 -> A는 1로 계산. 합계 16.
- [x] 버스트를 판단하는 기능

## 용어 사전

참고: [온라인 블랙잭 시뮬레이터](https://www.blackjacksimulator.net/ko/guide/blackjack-glossary/), 블랙잭 미션 설명

- deck: 카드 묶음
- hand: 카드 패
- dealer: 딜러
- player: 플레이어
- participant: 참여자. 딜러(dealer)와 플레이어(player)를 총칭
- dealout: 게임을 시작하며 카드 2장을 나눠주는 행위
- bust: 카드 합이 21을 초과해 패배하는 경우
- blackjack: 'dealout'에서 받은 카드 2장만으로 합계가 21이 된 경우
- push: 무승부
- hit: 플레이어가 카드를 한 장 더 요청
- stand: 추가 카드를 받지 않고 턴을 마침
- hole card: 딜러의 뒤집힌 카드
- up card: 딜러가 공개하는 첫 번째 카드
