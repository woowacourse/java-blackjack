# java-blackjack

블랙잭 미션 저장소

### 입력 요구사항

- 플레이어 입력
    - 쉼표(,)로 구분

- 플레이어별 히트/스탠드 입력
    - 히트 : y
    - 스탠드: n

### 출력 요구사항

- 초기 2장 출력
    - 형식: 숫자모양(3다이아몬드)
- 히트/스탠드 시 결과 출력
    - 소유중인 카드 전부 출력
- 딜러 규칙 출력
    - 16장 이하일 시 히트
- 게임 통계 출력
    - 딜러와 플레이어의 핸드와 총점 출력
- 게임 승패 출력
    - 딜러 a승 b무 c패
    - 플레이어 승/무/패

### 기능 요구사항

- 게임 시작 시 딜러와 플레이어들에게 카드 두 장씩 지급

- 카드 점수 계산
    - 2 ~ 10 → 숫자 그대로 계산
    - J, Q, K → 10점 계산
    - A -> 별도 처리

- 에이스(A) 처리
    - 1 or 11 (딜러와 플레이어에게 유리하게 합산)

- 히트/스탠드
    - 히트 시 카드 한 장 추가
    - 스탠드 시 턴 종료

- 딜러 규칙
    - 딜러의 핸드에 있는 카드의 총합이 16 이하일 경우 히트
    - 딜러의 핸드에 있는 카드의 총합이 17 이상일 경우 스탠드

- 버스트 판정
    - 핸드에 있는 카드의 숫자만 추출
    - 합이 21 초과시 버스트

- 카드 점수 총합 계산
    - 핸드의 모든 카드들의 숫자의 총합 반환

- 승패 판정
    - 딜러의 카드 점수 총합과 플레이어의 카드 점수 총합 비교
    - 플레이어의 승/무/패 여부 계산
    - 딜러는 플레이어의 총 승/무/패의 반대
