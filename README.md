# java-blackjack - 블랙잭 미션 저장소

## 기능 구현 목록

1. 게임 참여하는 사람은 쉽표 기준으로 입력 받는다.
2. 플레이어들의 베팅 금액을 입력 받는다.
3. 모두에게 카드 1장씩 2번 나눠준다.
4. 처음 받은 카드의 상태를 출력한다.
    1. 딜러는 첫번째 카드만 오픈
5. 플레이어턴 - 한명의 플레이어 옵션 모두 선택( 입력받은 순서대로)
    1. 플레이어가 버스트 상태이거나 n 입력 시 해당 플레이어 옵션 선태 종료
    2. 모든 입력 후에는 입력 후의 카드 상태 출력
        1. (예시와 다르지만, 실제 블랙잭 게임 처럼 모든 참가자가 해당 턴의 참가자 카드 상태를 볼 수 있어서 결정)
6. 딜러 턴
    1. 16이하인경우 - 카드 한장 추가 후 출력
        1. (예시와 다르지만, 실제 블랙잭 게임 처럼 모든 참가자가 딜러의 카드 상태를 볼 수 있어서 결정)
    2. 17 이상인경우 - 최종 상태 모두 출력(플레이어, 딜러)
7. 최종 수익 출력
