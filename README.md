# java-blackjack

블랙잭 미션 저장소

# 기능 명세서

---

## 입력

- [x]  참여자 이름을 입력받는다.
    - 게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)
- [x]  플레이어별로 베팅 금액을 입력받는다.
    - pobi의 배팅 금액은?
- [x]  플레이어별로 히트/스탠드를 입력받는다.
    - pobi는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)

## 출력

- [x]  카드 분배 기록을 출력한다.

   ```
   딜러와 pobi, jason에게 2장을 나누었습니다.
   딜러: 3다이아몬드
   pobi카드: 2하트, 8스페이드
   jason카드: 7클로버, K스페이드
   ```

- [ ]  플레이어의 히트/스탠드 여부와 관계없이 해당 플레이어가 보유한 카드 목록을 출력한다.

   ```
   pobi카드: 2하트, 8스페이드, A클로버
   ```

- [x]  딜러 카드 추가 여부를 출력한다.

   ```
   딜러는 16이하라 한장의 카드를 더 받았습니다.
   
   딜러는 17이상이라 카드를 받지 않았습니다.
   ```

- [ ]  참가자의 최종 보유 카드와 합계를 출력한다.

   ```
   딜러 카드: 3다이아몬드, 9클로버, 8다이아몬드 - 결과: 20
   pobi카드: 2하트, 8스페이드, A클로버 - 결과: 21
   jason카드: 7클로버, K스페이드 - 결과: 17
   ```

- [ ]  최종 수익을 출력한다.

   ```
   ## 최종 수익
   딜러: 10000
   pobi: 10000 
   jason: -20000
   ```


## 핵심 기능

- [x]  카드 덱 생성 및 셔플
- [x]  플레이어/딜러 관리
    - [x]  이름
    - [x]  가진 카드
    - [ ]  베팅 금액
- [x]  카드 점수 계산
- [x]  Ace 1/11 처리
- [x]  버스트 판정
- [x]  히트/스탠드 처리
    - [ ]  카드를 추가로 뽑아 21을 초과할 경우 배팅 금액을 잃는다.
- [x]  승패 판정
    - [x]  플레이어와 딜러의 합계가 같거나, 모드 버스트이면 무승부 처리한다.
    - [x]  플레이어의 합계가 21 이하이면서 플레이어의 합계가 딜러의 합계보다 크면 플레이어의 승리로 처리한다.
    - [x]  딜러의 합계가 21 초과이면서 플레이어의 합계가 21보다 작으면 플레이어의 승리로 처리한다.
    - [x]  플레이어의 합계가 딜러의 합계보다 작고 딜러의 합계가 21 이하이면 플레이어의 패배로 처리한다.
    - [x]  플레이어의 합계가 21 초과이면서 딜러의 합계가 21 이하이면 플레이어의 패배로 처리한다.
- [ ]  최종 수익 계산
- 플레이어 수익 계산
    - [ ]  플레이어와 딜러 모두 블랙잭: 0배
    - [ ]  플레이어 블랙잭: 베팅 금액의 1.5배
    - [ ]  플레이어 버스트: 베팅 금액의 -1.0배
    - [ ]  딜러 버스트: 베팅 금액의 1.0배
    - [ ]  플레이어 승리: 베팅 금액의 1.0배
    - [ ]  플레이어 패배: 베팅 금액의 -1.0배
    - [ ]  무승부: 0배
- 딜러 수익 계산
    - [ ]  `-1 * (플레이어 수익의 총 합)`

## 유효성 검사

### 이름

- [x]  빈 값을 입력한 경우
    - [ERROR] 빈 값을 입력할 수 없습니다.
- [x]  1 ~ 10자 범위를 벗어나는 경우
    - [ERROR] 이름은 1 ~ 10자 내여야 합니다.
- [ ]  이름이 다른 플레이어와 중복되는 경우
    - [ERROR] 이름은 중복될 수 없습니다.

### 히트/스탠드

- [x]  y 또는 n이 아닌 값을 입력한 경우
    - [ERROR] 입력은 y 또는 n으로만 입력해야 합니다.

### 베팅 금액

- [ ]  음수 또는 0인 경우
    - [ERROR] 베팅 금액은 양수여야 합니다.
- [ ]  1,000원 단위가 아닌 경우
    - [ERROR] 베팅 금액은 1,000원 단위여야 합니다.
- [ ]  1,000원 ~ 100,000원 범위를 벗어난 경우
    - [ERROR] 베팅 금액은 1,000원에서 100,000원 사이여야 합니다.
