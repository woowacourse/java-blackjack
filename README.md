# java-blackjack

블랙잭 미션 저장소

## 기능 요구 사항

#### 플레이어 이름 입력 기능

- [x] 안내 메세지 출력 후 참여할 플레이어 이름들을 입력을 받는다
    - [x] 널값이나 빈 공백 예외처리
    - [x] 쉼표 기준으로 입력 분리
     
- Player
  - Name
    - [x] 이름이 공백이면 예외처리
    - [x] 이름이 10글자 이상이면 예외처리
  - Hand
    - [ ] 보유한 카드의 점수 총합을 계산한다.
      - [x] 카드의 숫자 계산은 카드 숫자를 기본으로 한다.
      - [x] Ace는 1 또는 11로 계산한다.
      - [ ] J, Q, K는 각각 10으로 계산한다.

- Players
  - [ ] 중복된 닉네임은 예외처리
  - [ ] 플레이어 이름이 7개 초과시 예외처리

#### 게임 진행

- [ ] 게임 시작 준비
    - [x] 카드 덱을 만든다.
        - [x] 카드덱은 52개의 카드로 구성된다.
    - [x] 카드덱은 무작위의 순서를 가진다.
    - [ ] 딜러와 각 플레이어는 두 장의 카드를 지급받는다.
    - [ ] 카드 배정 결과 출력

- [ ] 각 플레이어의 핸즈의 총합이 21을 넘는지 검사한다.
    - [ ] 2~10은 숫자 그대로 계산한다.
    - [ ] Ace는 1또는 11로 계산한다.
    - [ ] J, Q, K 는 10으로 계산한다.
  - [ ] 21 이 넘지 않는 경우, 추가로 카드를 뽑을 수 있다.

- [ ] 플레이어 별 추가 카드 오픈 여부 선택 기능
    - [ ] y선택시, 플레이어 핸즈 출력
    - [ ] n이 나오거나, 버스트가 될 때까지 반복
- [ ] 딜러 차례
    - [ ] 딜러 핸즈 총합이 16이하일 떄
        - [ ] 딜러에게 카드 한 장 draw
- [ ] 게임 결과 출력 기능
- [ ] 최종 승패 출력 기능

```
실행 결과
게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)
pobi,jason

y
```
