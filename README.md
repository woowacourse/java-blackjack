# java-blackjack

## 구현할 기능
- [x] 플레이어와 딜러에게 무작위 카드를 지급한다.
- [x] Ace 카드 값을 1 또는 11로 조정한다.
- [x] 카드 총 합계를 계산한다.
- [x] 플레이어의 버스트 여부를 판단한다.
- [x] 딜러가 카드를 더 뽑아야 하는지 판단한다.
- [x] 플레이어별 승패 여부를 판단한다.

## 기능 요구사항
- 블랙잭 게임을 구현한다.
- 플레이어는 게임을 시작할 때 베팅 금액을 정해야 한다.
- 카드의 숫자 계산은 카드 숫자를 기본으로 한다. 예외로 Ace는 1 또는 11로 계산할 수 있다.
- King, Queen, Jack은 각각 10으로 계산한다.
- 게임을 시작하면 플레이어와 딜러는 두 장의 카드를 지급받는다.
- 카드 숫자가 21을 넘지 않을 경우, 원한다면 카드를 계속 뽑을 수 있다.
- 카드 숫자가 21을 초과할 경우 패배하여 베팅 금액을 모두 잃는다.
- 처음 두 장의 카드 합이 21일 경우 블랙잭이 된다. 승리시 베팅 금액의 1.5배를 받는다.
- 딜러는 카드 합이 16 이하이면 반드시 카드를 뽑아야 한다.

## 실행 결과
```
게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)
pobi,jason

pobi의 배팅 금액은?
20000

jason의 배팅 금액은?
10000

딜러와 pobi, jason에게 2장을 나누었습니다.
딜러카드: J다이아몬드
pobi카드: 9다이아몬드, 10하트
jason카드: K다이아몬드, 7하트

pobi은 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)
y
pobi카드: 9다이아몬드, 10하트, 8하트
jason은 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)
n

딜러카드: J다이아몬드, Q다이아몬드 - 결과: 20
pobi카드: 9다이아몬드, 10하트, 8하트 - 결과: 27
jason카드: K다이아몬드, 7하트 - 결과: 17

## 최종 수익
딜러: 30000
pobi: -20000
jason: -10000
```
