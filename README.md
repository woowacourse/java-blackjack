# java-blackjack

# 입력

✅ 게임 참가자 입력 뷰

```text  
게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)  
pobi,jason  
```  

---  

✅ 카드 수령 여부 입력 뷰

```text  
pobi는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)y  
```  

# 출력

✅ 게임 참가자 출력 뷰

```text  
딜러와 pobi, jason에게 2장을 나누었습니다.  
```  

---  

☑️ 게임 참가자, 딜러 카드 분배 결과 뷰

```text  
딜러카드: 3다이아몬드  
pobi카드: 2하트, 8스페이드  
jason카드: 7클로버, K스페이드  
```  

---  

☑️ 게임 참가자 전체 카드 출력 뷰

```text  
pobi카드: 2하트, 8스페이드, A클로버  
```  

---  

☑️ 딜러 카드 수령 여부 출력 뷰

```text  
딜러는 16이하라 한장의 카드를 더 받았습니다.  
```  

---  

☑️ 참가자 및 딜러 최종 결과 출력 뷰

```text  
딜러카드: 3다이아몬드, 9클로버, 8다이아몬드 - 결과: 20  
pobi카드: 2하트, 8스페이드, A클로버 - 결과: 21  
jason카드: 7클로버, K스페이드 - 결과: 17  
```  

---  

☑️ 최종 승패 출력 뷰

```text  
## 최종 승패  
딜러: 1승 1패  
pobi: 승 jason: 패  
```  

# 핵심 기능

☑️ 게임 참가자 입력 기능

- 게임에 참여할 사람의 이름을 입력 받는다.
- 게임 참가자 이름의 좌우 공백은 제거한다.
- 사용자의 입력은 ,(쉼표)로 구분한다.

⚠️ 기능 예외 사항

- 게임 참가 인원은 2~8명이 아니라면 예외를 발생 시켜야 한다.
- 게임 참가자 이름은 공백을 포함한 2~5글자가 아니라면 예외를 발생 시켜야 한다.
- 입력값에 공백만 입력되면 예외를 발생 시켜야 한다.

---  

☑️ 카드 무작위 섞기 기능

- 사용하는 카드를 무작위로 섞는다.
- 카드는 총 6벌이다.

---  

☑️ 카드 뽑기 기능

- 카드를 한 장 뽑는 기능입니다.
- 카드는 총 6벌이다.
- 중복될 수 있는 카드는 6개이다.
- 각 카드마다 6장을 넘어서면, 더 이상 출현되어선 안된다.

---  

☑️ 카드 점수 계산 기능

- 카드의 점수를 계산하는 기능입니다.

---  

☑️ 딜러 카드 추가 수령 여부 판단 기능

- 딜러는 처음에 받은 2장의 합계가 16이하이면 반드시 1장의 카드를 추가로 받아야 하고, 17점 이상이면 추가로 받을 수 없다.

---  

☑️ 최종 승패 계산 기능

- 게임 참가자의 승패 여부를 계산합니다.
