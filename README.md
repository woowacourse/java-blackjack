# java-blackjack

블랙잭 미션 저장소

## 입력

- [x] 게임에 참여할 사람의 이름 입력 받기(구분자 ,)
    - [x] 공백일 경우 `IllegalArgumentException`을 발생시킨다.

- [] 플레이어에게 배팅 금액 입력 받기
    - [] 숫자 이외의 문자를 입력할 경우 `IllegalArgumentException`을 발생시킨다.
    - [] 공백일 경우 `IllegalArgumentException`을 발생시킨다.

- [x] 플레이어에게 hit 여부 입력 받기(y,n).
    - [x] 공백일 경우 `IllegalArgumentException`을 발생시킨다.

## 비지니스 기능

- 플레이어 등록
    - [x] 최소 2자 ~ 최대 7자, 이외 범위는 `IllegalArgumentException`을 발생시킨다.
    - [x] 영어&한글만 가능, 이외 형식은 `IllegalArgumentException`을 발생시킨다.
    - [x] 중복된 이름이 존재할 경우 `IllegalArgumentException`을 발생시킨다.

- 배팅 등록
    - [] 배팅 금액은 양수 값만 입력으로 받는다.
    - [] 배팅 금액은 1,000원 이상 1,000,000원 이하, 이외 범위는 `IllegalArgumentException`을 발생시킨다.

- 게임 초기화
    - [x] 게임 시작 시, 모든 카드를 덱에 등록하고 셔플한다.
    - [x] 딜러와 플레이어에게 카드를 각각 2장씩 배분한다.
      - [ ] 초기 2장의 카드 합이 21인 경우 블랙잭(Blackjack)으로 판정한다.

- 카드 점수 합산
    - [x] 카드 합산 점수를 계산한다.
        - [x] Ace 점수(1/11)를 처리한다.

- 게임 진행 판정
    - [x] 딜러와 플레이어의 히트/스탠드 여부를 판정한다.
    - [x] Bust 여부(21 초과)를 판정한다.

- 승패 판정
    - [x] 플레이어 승패 여부를 판정한다.
    - [x] 딜러의 승패 전적을 계산한다.

- 수익 계산 
    - [ ] 플레이어의 수익률 결정:
      - 블랙잭 승리: 배팅 금액의 1.5배 (단, 딜러와 동시 블랙잭이면 무승부)
      - 일반 승리: 딜러보다 점수가 높거나, 딜러가 Bust일 때 배팅 금액의 1.0배
      - 무승부(Push): 딜러와 점수가 같을 때 수익 0
      - 패배: 배팅 금액 전액 손실 (-1.0배)
    - [ ] 딜러의 최종 수익 계산: 모든 플레이어의 수익 합산 결과의 역산

## 출력

- [x] 이름과 함꼐 게임 시작 문구 출력
- [x] 가지고 있는 카드 목록 출력
    - [x] 딜러는 맨 처음 받은 한 장은 출력 x
- [x] 카드 합산 결과 출력
- [x] 딜러 및 각 플레이어별 최종 승패 결과 출력
- [] 딜러 및 각 플레이어별 수익 결과 출력
