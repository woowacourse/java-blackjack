# 블랙잭 미션 저장소

## 용어 정의 및 모델링

### 카드 (Card)

| 용어         | 설명                                         |
|------------|--------------------------------------------|
| Card       | 블랙잭에 사용되는 카드. 값과 문양을 가진다.                  |
| Card Value | 카드의 값. A, 2, 3, ..., 10, J, Q, K가 있다.      |
| Card Shape | 카드의 문양. DIAMOND, HEART, CLOVER, SPADE가 있다. |

### 덱 (Deck)

| 용어   | 설명                                    |
|------|---------------------------------------|
| Deck | 게임에 필요한 카드들. 블랙잭 카드가 중복없이 총 52장 존재한다. |

### 참여자 (Participant)

| 용어               | 설명                                                         |
|------------------|------------------------------------------------------------|
| Participant      | 게임에 참여하는 사람. 고유한 이름을 가지며 규칙에 따라 덱에서 카드를 뽑는다. 플레이어이거나 딜러이다. |
| Participant Name | 게임에 참여하는 사람의 이름. 이름은 1~5자 이다. 각 참여자들끼리 이름에 중복이 없어야 한다.     |

### 딜러 (Dealer)

| 용어     | 설명                           |
|--------|------------------------------|
| Dealer | 플레이어의 승패를 결정짓는 카드를 가지고있는 사람. |

### 플레이어 (Player)

| 용어     | 설명                                           |
|--------|----------------------------------------------|
| Player | 게임에 참여하는 사람. 고유한 이름을 가지며 규칙에 따라 덱에서 카드를 뽑는다. |

### 카드를 뽑는 규칙 (Rule)

| 용어            | 설명                                                           |
|---------------|--------------------------------------------------------------|
| Rule          | 참여자가 카드를 뽑는 규칙. 각 참여자는 카드를 뽑지 않을때 까지 규칙을 반복하고, 다음 참여자로 넘어간다. |
| Player's Rule | 카드의 합이 21미만일 때, 추가로 카드를 뽑을 것인지 선택할 수 있다.                     |
| Dealer's Rule | 카드의 합이 17미만일 때, 반드시 카드를 뽑는다. 카드의 합이 17이상일 때, 카드 뽑기를 멈춘다.     |

### 상태 (State)

| 용어        | 설명                    |
|-----------|-----------------------|
| State     | 카드를 가진 참여자가 될 수 있는 상태 |
| Stay      | 카드를 더 뽑지 않는 경우        |
| Hit       | 카드를 더 뽑는 경우           |
| Burst     | 카드 총합이 21을 초과할 경우     |
| Blackjack | 카드 총합이 21인 경우         |

### 수익 (Profit)

| 용어     | 설명                         |
|--------|----------------------------|
| Profit | 플레이어가 얼마나 벌었는지를 금액으로 나타낸 것 |

### 베팅 (Betting)

| 용어             | 설명                                      | 
|----------------|-----------------------------------------|
| Betting        | 플레이어가 승패에 거는 돈                          |
| Betting Amount | 베팅에 거는 돈의 양 (최소 1,000원 ~ 최대 1,000,000원) |

## 기능 요구 사항

> 블랙잭 게임을 변형한 프로그램을 구현한다. <br> 블랙잭 게임은 딜러와 플레이어 중 카드의 합이 21 또는 21에 가장 가까운 숫자를 가지는 쪽이 이기는 게임이다.

- [x] 카드의 숫자 계산은 카드 숫자를 기본으로 한다.
    - [x] 예외로 Ace는 1 또는 11로 참여자에게 유리하도록 계산할 수 있다.
    - [x] King, Queen, Jack은 10으로 계산한다.

- [x] 게임을 시작하면 플레이어는 두 장의 카드를 지급 받는다.
- [x] 두 장의 카드 숫자를 합쳐 21을 초과하지 않으면서 21에 가깝게 만들면 이긴다.
    - [x] 21을 넘지 않을 경우 원한다면 얼마든지 카드를 계속 뽑을 수 있다.

- [x] 딜러는 처음에 받은 2장의 합계가 16이하이면 반드시 1장의 카드를 추가로 받아야 한다. 17점 이상이면 추가로 받을 수 없다.
- [x] 게임을 완료한 후 각 플레이어별로 승패를 출력한다.

## 출력 예시

```

10000

jason의 배팅 금액은?
20000

딜러와 pobi, jason에게 2장을 나누었습니다.
```

## 게임 흐름 세부 조건 사항

### 게임 준비 (Player Name 입력 단계)

- [x] 공백 입력은 허용하지 않는다.
- [x] 쉼표를 기준으로 Player Name을 분리한다.
- [x] 5자를 넘는 Player Name은 허용하지 않는다.
- [x] Player는 최대 5명이다.

### 게임 시작 (베팅, Deck 셔플 및 Card 세팅 단계)

- [x] Player는 게임을 시작할 때 Betting Amount을 정해야 한다.
    - [x] Betting Amount는 최소 1,000원 ~ 최대 1,000,000원 이다.
    - [x] Betting Amount는 숫자로만 입력이 가능하다.

- [x] Deck은 Shuffle할 수 있다.
- [X] Participants에게 Card를 두 장씩 나눠준다.

### 게임 중 (Rule에 따라 각 Player가 Card를 뽑는 단계)

- [x] 각 Player는 Card를 더 받을지 말지 선택할 수 있다.
    - [x] 카드를 추가로 뽑아 Burst가 될 경우 Betting Amount를 모두 잃게 된다.
    - [x] Stay나 Burst가 나올 경우 다음 Player로 넘어간다.
    - [x] 만약 다음 Player가 없을 경우 다음 단계로 넘어간다.

- [x] 모두 Burst 상태인 경우 결과를 보여준다.
- [x] 아니라면 Dealer는 Card를 뽑는다.
    - [x] Dealer는 Card 합계가 17점 이상일 때 까지 카드를 받는다.

- [x] Participants의 게임 결과를 보여준다.
    - 게임 결과는 각 Participant가 뽑은 Card와 총 합을 보여주는 것이다.

### 게임 종료 (Dealer와 Player의 State 또는 Card 총합을 비교하고 출력하는 단계)

- [x] Participant의 승패 정보를 확인하고 최종 수익을 출력한다.
    - [x] Dealer가 Burst일 때
        - Player가 Burst면 Player 패, Dealer 승
        - Player가 Stay면 Player 승, Dealer 패

    - [x] Dealer가 Stay일 때
        - Player가 Burst면 Player 패, Dealer 승
        - Player가 Stay면 Card 총합을 비교하여 승패를 비교한다.
            - Card 총합이 같다면 무승부이다. (모두 돈을 잃지 않았기 때문이다.)

- [x] 승자는 Betting Amount 만큼 수익을 얻는다.
- [x] 패자는 Betting Amount를 모두 잃는다.
- [x] Dealer는 Player의 승패에 따라 Profit이 결정된다.
    - `-1 * (Sum of Players's Profit)`을 Dealer의 Profit으로 한다.

- [x] Player가 Blackjack이면 Betting Amount의 1.5배를 Profit으로 얻는다.
