# java-blackjack

블랙잭 미션 저장소
## 기능 구현 목록

## 정한 규칙
- 덱은 1개로 제한한다. (카드 56장 중복 x)
- 첫 카드합이 21이면 블랙잭이 되었음을 알린다.
- 21이 될경우 추가로 카드를 받지 않는다.
- A는 버스트일 경우 1로 계산되고 아닐 경우 11로 계산된다.
    - 3개 이상일 경우 테스트 케이스 생각해봐야함.

### 블랙잭
- 이름을 저장한다.
    - 1~7자이다.
    - 영문만 입력받는다.
- 받은 카드를 저장한다.
  - A의 1or11 처리를 한다.    **주의**
- 카드의 숫자를 반환한다. (J/Q/K - 10)
- 카드의 총합을 리턴한다.
- 버스트인지 확인하다 (21 초과하는지 확인한다.)

### 딜러
- 17 이상이 될때까지 무조건 카드를 받는다. **주의**

### 덱
- 56장의 카드를 저장한다.
    - 카드를 리스트로 관리한다.
- 맨 위에 있는 카드를 가져올 수 있다.(pop)
    - 가져간 카드는 다시 나오지 않는다.

### 카드관리
- ENUM(문양), ENUM(숫자, JKQ, A)로 각각 저장한다. 

### 카드 생성기
- ENUM(문양), ENUM(숫자, J/K/Q, A) 두 ENUM 각각을 조합해서 56개의 카드를 생성한다. - 카드 객체

### 카드
- 문양, 숫자


### 셔플기
- 덱을 받아 셔플한다.


### 입출력
- 플레이어 이름은 `,`로 구분한다.
