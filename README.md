## 용어 설명
- **피치(Pitch)**: 딜러가 카드를 플레이어에게 나눠주는 행위
- **히트(Hit)**: 참여자가 카드를 더 받는 상태
- **스테이(Stay)**: 참여자가 카드를 더이상 받지 않는 상태
- **버스트(Burst)**: 숫자 카드의 합이 21을 초과하는 경우
- **액션(Action)**: 딜러/플레이어가 차례에 내리는 모든 선택
## 기능 목록

### 참여 플레이어 닉네임 입력 기능
- 게임에 참여할 플레이어들의 닉네임을 입력받습니다.
```angular2html
게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)
pobi,jason
```
- **[예외 상황]**
  - 각 닉네임의 길이가 `4`~`10`자 범위가 아닌 경우
  - 플레이어 인원수가 `1`~`7`명 범위가 아닌 경우
  - 플레이어 닉네임 입력 구분이 `,`가 아닌 경우

### 딜러 피치 기능
- 딜러와 모든 플레이어에게 카드 2장씩 랜덤하게 나눠줍니다.
- 딜러와 모든 플레이어이 가진 모든 카드를 출력합니다.
- 단, 딜러의 카드는 **한 장**만 출력됩니다.
```angular2html
딜러와 pobi, jason에게 2장을 나누었습니다.
딜러카드: 3다이아몬드
pobi카드: 2하트, 8스페이드
jason카드: 7클로버, K스페이드
```

### 플레이어 액션 진행 기능
- 모든 플레이어가 스테이 또는 버스트 상태가 될 때까지 턴을 진행합니다.
- 각 플레이어는 자신의 턴에서 카드를 더 받을지에 대해 결정합니다.
```angular2html
pobi는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)
```
  - `y`를 고른 경우(히트를 고른 경우): 카드를 한장 더 받습니다.
  - `n`을 고른 경우(스테이를 고른 경우): 카드를 더이상 받지 않고 차례를 마칩니다.
- 각 플레이어의 턴이 끝날 때마다 해당 플레이어의 카드 상태를 출력합니다.
```angular2html
pobi카드: 2하트, 8스페이드, A클로버
```
- **[예외 상황]**
  - 히트 또는 스테이 여부에 대한 사용자 답변이 `y` 또는 `n`이 아닌 경우
### 딜러 액션 진행 기능
- 딜러의 핸드의 카드 합이 17 이상일 때 까지 카드를 뽑습니다.
- 따라서, 카드 합이 17 미만인 경우에는 자동으로 히트, 이상일 경우에는 스테이로 딜러 상태가 자동으로 결정됩니다.
- 딜러가 카드를 한장 더 받는 경우, 다음의 문구가 출력됩니다.
```angular2html
딜러는 16이하라 한장의 카드를 더 받았습니다.
```

### 게임 결과 안내 기능
- 딜러와 모든 플레이어의 카드 상태, 점수 및 승패 결과를 출력합니다.
```angular2html
딜러카드: 3다이아몬드, 9클로버, 8다이아몬드 - 결과: 20
pobi카드: 2하트, 8스페이드, A클로버 - 결과: 21
jason카드: 7클로버, K스페이드 - 결과: 17

## 최종 승패
딜러: 1승 1패
pobi: 승 
jason: 패
```
- 게임 승패 계산표

  | 딜러  | 플레이어 |    플레이어 승패     |
  |:---:|:----:|:--------------:|
  | 버스트 | 버스트  |       패배       |
  | 버스트 | 스테이  |       승리       |
  | 스테이 | 버스트  |       패배       |
  | 스테이 | 스테이  | 점수가 더 높은 쪽이 승리 |

## 추가 기능 구현 목록
- 모든 플레이어의 베팅 금액 입력 기능
  - 예외 처리:
    - 베팅 금액 범위가 5,000 ~ 100,000,000이 아닌 경우
    - 베팅 금액 단위가 1,000이 아닌 경우
- 베팅 결과 금액 계산 기능
  - 승리: 원금의 2배
  - 블랙잭으로 인한 승리: 원금의 1.5배
  - 무승부: 원금의 1배
  - 패배: 원금의 0배
