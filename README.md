# java-blackjack - 블랙잭 미션 저장소


## 요구사항 분석 및 이해
- 명사 개념찾기
    - 사람(플레이어, 딜러)
        - 이름
        - 결과
    - 카드
        - 이름
        - 점수
    - 최종승패
- 동사 행동 찾기
    - 이름 입력 받기
    - 카드 나눠주기
    - 플레이어 버스트 상태인지 확인
    - 옵션 선택하기
        - 히트
        - 스탠드
    - 플레이어 상태 출력
    - 딜러 현재 점수 확인
    - 딜러 한장 더 받기
    - 딜러 버스트 상태인지 확인
    - 최종 플레이어와 딜러 상태 출력
    - 최종 승패 계산하기
    - 최종 승패 출력하기
- 업무규칙
    - `목표`
        - 카드 합을 21에 최대한 가깝게 만들기
        - 단, 21을 넘기면 즉시 패배(버스트)
        - 플레이어 vs 딜러 구조
    - `카드 정책`
        - 총 52장(조커 제외)
        - 4종류의 무늬가 13개씩 (2~10) + JQK + A
        - 카드는 3개의 종류가 있다.
            - 숫자 카드 : 2~10 (숫자 그대로)
            - 영문 카드 :JQK (10점)
            - 에이스 : A(1 or 11점 중 유리한 쪽 자동 선택)
    - `게임 진행`
        - 플레이어 & 딜러 각각 **카드 2장** 받음
            - 딜러는 **1장 오픈, 1장 히든**
        - 플레이어 차례
            - **Hit:** 카드 1장 더 받기
            - **Stand:** 더 안 받고 멈추기
            - **~~Double Down:** 배팅 2배 + 카드 1장만 받고 종료~~
            - **~~Split:** 같은 카드 2장이면 나눠서 2판 진행~~
        - 딜러 차례
            - 합이 **16 이하 → 무조건 Hit, 17 이상 → Stand**
    - `승패 기준`
        - 플레이어가 21 초과 → ❌ 패배
        - 딜러가 21 초과 → ✅ 플레이어 승
        - 둘 다 21 이하:
            - **합이 더 높은 쪽 승**
            - 같으면 **무승부(Push)**
    - `블랙잭`
        - 처음 받은 2장이 A + J,Q,K ⇒ 1.5배 배당 !

### 유즈케이스

1. 게임에 참여할 사람의 이름을 입력받는다.
2. 모두에게 카드 1장씩 2번 나눠줌
    1. 딜러는 첫번째 카드만 오픈
3. 플레이어턴 - 한명의 플레이어 옵션 모두 선택( 입력받은 순서대로)
    1. 플레이어가 버스트인지 확인.
    2. y 선택시 무조건 상태 출력
    3. n 선택시
        1. 카드 2장 소유면 출력
        2. 이외에는 노출력
4. 딜러 턴
    1. 16이하인경우
        1. 카드 한장 추가 후 출력
    2. 17 이상인경우
        1. 최종 상태 모두 출력(플레이어, 딜러)
5. 최종 승패 결과


## 기능 구현 목록
1. 게임 참여하는 사람은 쉽표 기준으로 입력 받는다.
2. 딜러가 자신과 플레이어들에게 카드를 나눠준다.
3. 처음 받은 카드의 상태를 출력한다.
    1. 딜러가 두 번째 받은 카드는 출력 안한다.
4. 플레이어 턴이 시작된다.
    1. 각 플레이어마다 옵션을 연속해서 선택한다.
        1. 플레이어가 버스트가 아닌 경우
            1. 히트(y) 선택 - 상태 출력
            2. 스탠드(n) 선택
                1. 카드 2장 소유한 상태면 출력
                2. 이외에는 미출력
        2. 버스트인 경우
            1. 결과(승/무/패) 상태 갱신
            2. 최종 상태 출력
5. 딜러턴이 시작된다.
    1. 딜러의 총 점수 확인
        1. 16이하인 경우 - 카드 한장 추가 후 출력
        2. 16 초과인 경우 - 최종 (플레이어, 딜러) 상태 모두 출력
6. 최종 승패 결과 출력
