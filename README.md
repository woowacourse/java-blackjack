# java-blackjack

블랙잭 미션 저장소

# 기능 명세서

## 입력
- [ ]  참여 인원 이름을 입력받는다.
    - [ ]  게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)
- [ ]  플레이어별로 히트/스탠드 입력받기
    - [ ]  pobi는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)

## 출력
- [ ]  카드 분배 기록을 출력한다.
   ```java
   딜러와 pobi, jason에게 2장을 나누었습니다.
   딜러카드: 3다이아몬드
   pobi카드: 2하트, 8스페이드
   jason카드: 7클로버, K스페이드
   ```
- [ ]  플레이어 보유 카드를 출력한다.
   ```java
   pobi카드: 2하트, 8스페이드, A클로버
   ```
- [ ]  딜러 카드 추가 여부 출력
   ```markdown
   딜러는 16이하라 한장의 카드를 더 받았습니다.
   
   딜러는 17이상이라 카드를 받지 않았습니다.
   ```
- [ ]  참가자의 결과를 최종 보유 카드와 합계를 출력한다.
   ```markdown
   딜러카드: 3다이아몬드, 9클로버, 8다이아몬드 - 결과: 20
   pobi카드: 2하트, 8스페이드, A클로버 - 결과: 21
   jason카드: 7클로버, K스페이드 - 결과: 17
   ```
- [ ]  최종 승패를 출력한다
   ```markdown
   딜러: 1승 1패
   pobi: 승 
   jason: 패
   ```

## 핵심 기능

- [x] 카드 점수 계산
- [x]  Ace 1/11 처리
- [x]  버스트 판정
- [ ]  히트/스탠드 처리
- [x]  승패 판정

## 유효성 검사
### 이름
- [ ]  빈 값을 입력한 경우
- [ ]  1 ~ 10자 범위를 벗어나는 경우
- [ ]  이름이 다른 플레이어와 중복되는 경우
### 히트/스탠드
- [ ]  y 또는 n이 아닌 값을 입력한 경우
