# 블랙잭

## 구현할 기능 목록

### 카드
- [ ] 카드는 무늬(Suit)와 숫자(Rank)를 가진다
- [ ] 무늬는 스페이드, 하트, 다이아몬드, 클로버 4가지이다
- [ ] 숫자는 A, 2~10, J, Q, K이며 A는 1 또는 11로 계산한다

### 덱
- [ ] 52장의 카드로 구성된다
- [ ] 게임 시작 시 카드를 섞는다
- [ ] 카드를 한 장씩 뽑을 수 있다

### 참여자
- [ ] 딜러와 플레이어는 카드를 받을 수 있다
- [ ] 딜러와 플레이어는 보유한 카드의 점수를 계산할 수 있다
- [ ] 플레이어는 21 미만이면 카드를 더 받을 수 있다
- [ ] 딜러는 16 이하이면 카드를 더 받아야 한다
- [ ] ACE를 포함할 때 버스트가 나지 않으면 11로, 아니면 1로 계산한다

### 게임 진행
- [ ] 게임 시작 시 딜러와 플레이어에게 각 2장씩 나눠준다
- [ ] 초기 딜 시 딜러는 카드 1장만 공개한다
- [ ] 플레이어는 순서대로 히트(y) 또는 스테이(n)를 선택한다
- [ ] 플레이어가 히트 시 카드를 받고 현재 카드를 출력한다
- [ ] 플레이어가 처음부터 스테이 시 초기 카드를 출력한다
- [ ] 플레이어가 히트 후 스테이 시 카드를 출력하지 않는다
- [ ] 모든 플레이어 턴 종료 후 딜러 턴을 진행한다

### 결과 계산
- [ ] 딜러와 플레이어의 최종 점수를 비교해 승패를 결정한다
- [ ] 버스트(21 초과) 시 해당 참여자가 패배한다
- [ ] 딜러 결과는 플레이어 결과의 역으로 계산한다

### 입출력
- [ ] 참여자 이름을 쉼표 기준으로 입력받는다
- [ ] 히트 여부를 y/n으로 입력받는다
- [ ] 초기 딜 결과를 출력한다
- [ ] 최종 카드 및 점수를 출력한다
- [ ] 최종 승패를 출력한다
