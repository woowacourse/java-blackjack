  - view: 입력을 받을 때, 형식에 맞지 않는 것은 입력 자체가 되지 않게 제한하는 경우를 생각하면 view에서 해도 되는가?
  - dto나 view 가 아니라면 controller?, 애그리거트 진입점의 정적 팩토리 매서드?
