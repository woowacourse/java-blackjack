# java-blackjack
## 블랙잭 미션 저장소



## 기능 구현 목록


1. 게임 참여하는 사람은 쉽표 기준으로 입력 받는다.
2. 딜러가 자신과 플레이어들에게 카드를 나눠준다.
3. 처음 받은 카드의 상태를 출력한다.
    1. 딜러가 두 번째 받은 카드는 출력 안한다.
4. 플레이어 턴이 시작된다.
    1. 각 플레이어마다 옵션을 연속해서 선택한다.
        1. 플레이어가 버스트가 아닌 경우
            1. 히트(y) 선택 - 상태 출력
            2. 스탠드(n) 선택
                1. 카드 2장 소유한 상태면 출력
                2. 이외에는 미출력
        2. 버스트인 경우
            1. 결과(승/무/패) 상태 갱신
            2. 최종 상태 출력
5. 딜러턴이 시작된다.
    1. 딜러의 총 점수 확인
        1. 16이하인 경우 - 카드 한장 추가 후 출력
        2. 16 초과인 경우 - 최종 (플레이어, 딜러) 상태 모두 출력
6. 최종 승패 결과 출력
