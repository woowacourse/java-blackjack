# Java Blackjack 사이클2

규칙을 적용하면서 블랙잭에 베팅 기능을 추가한다.
---

## 추가 기능 목록

- [x] 플레이어는 게임을 시작할 때 배팅 금액을 정해야 한다.
- [x] 첫 두 장이 합이 21이 되면 배팅 금액의 1.5배를 딜러에게 받는다.
- [x] 첫 두 장이 딜러와 플레이어가 동시에 블랙잭일 경우엔 배팅한 금액을 돌려받는다.
- [x] 게이머가 카드를 뽑아 21을 초과할 경우 배팅 금액을 모두 잃는다.
- [x] 딜러가 21을 초과하면 그 시점까지 남아 있는 플레이어들은 패에 상관 없이 승리한다.
