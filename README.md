## 구현 내용

### 필수 요구 사항 체크리스트

- [x] 블랙잭 게임은 딜러와 플레이어 중 카드의 합이 21 또는 21에 가장 가까운 숫자를 가지는 쪽이 이긴다.
- [x] Ace는 1 또는 11로 계산한다.
- [x] King, Queen, Jack은 각각 10으로 계산한다.
- [x] 게임을 시작하면 플레이어는 두 장의 카드를 지급 받는다.
- [x] 21을 넘지 않을 경우 원한다면 얼마든지 카드를 계속 뽑을 수 있다.
- [x] 딜러는 처음에 받은 2장의 합계가 16이하이면 반드시 1장의 카드를 추가로 받아야 한다.
- [x] 딜러는 처음에 받은 2장의 합계가 17점 이상이면 추가로 받을 수 없다.
- [x] 게임을 완료한 후 각 플레이어별로 승패를 출력한다.

---

- [ ] 플레이어는 게임 시작시 베팅 금액을 입력 받는다.
- [ ] 카드를 뽑았을 때 21을 초과하게 되면 베팅 금액을 _모두_ 잃는다.
- [ ] 처음 두 장의 카드 합이 21 (블랙잭)인 경우 베팅 금액의 1.5배를 받는다.
- [ ] 딜러와 참가자 둘 다 블랙잭이면 참가자는 베팅 금액을 돌려 받는다.

### 추가로 정의한 제약 사항

**사이클 1**
1. 플레이어가 모두 bust라면, dealer는 추가 카드를 받지 않고 무조건 승리한다.
2. 플레이어 카드 bust라면, 추가 카드를 받지 못하고 턴을 넘긴다.

**사이클 2**
1. 베팅 금액은 1원부터 된다.
