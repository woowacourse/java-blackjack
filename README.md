# java-blackjack

블랙잭 미션 저장소

### 기능 구현 목록

- 딜러
  - [ ] 처음 카드를 두 장 받는다.
  - [ ] 카드를 한장만 공개한다. (제약 : 처음만 공개)
  - [ ] 두 카드의 합이 16이하일때 카드를 한장 더 받는다.

- 플레이어
  - [ ] 처음에 카드를 두 장 받는다.
  - [ ] 카드를 두장 전부 공개한다.
  - [ ] 카드의 합이 21을 넘지 않을 경우, 카드를 뽑을 수 있다.
  - [ ] 플레이어의 이름은 겹치면 안된다.
  - [ ] 플레이어는 한명 이상이어야 한다.
  
- 랜덤 카드 생성
  - [ ] 카드는 2~9,J,Q,K,A의 넘버링을 가진다.
  - [ ] 카드는 하트, 다이아몬드, 스페이드, 클로버의 문양을 가진다.
  - [ ] 카드는 중복되면 안된다.

- 점수 계산
  - [ ] 카드는 숫자 넘버링일 경우 그 숫자로, J,K,Q의 넘버링일때는 10점의 점수를 가진다.
  - [ ] A는 카드에 따라 1 또는 11의 점수를 가질 수 있다.
    - [ ] A가 없을 경우, 카드의 합이 10이하일 경우에는 11점, 11점 이상일때는 1점을 가진다.
    - [ ] A가 이전에 있을경우 그 A는 무조건 1점을 가진다.
  - [ ] 플레이어의 점수는 카드의 합산 점수로 결정된다.

- 승자, 버스트 판정
  - [ ] 21점을 넘으면 버스트로 판정되고 게임은 패배한다.
  - [ ] 그렇지 않을 경우, 딜러보다 점수가 높은 사람은 승리한다.

- 입력
  - [ ] 플레이어 이름을 입력한다.
    - [ ] 이름은 쉼표 단위로 구분한다.
  - [ ] 플레이어가 카드를 뽑고 싶을 때 y,n만을 입력한다.
    - [ ] 다른 문자가 들어왔을 때 예외를 뱉는다.

- 출력
  - [ ] 딜러의 첫번째 카드를 출력한다.
  - [ ] 플레이어의 카드 덱을 출력한다.
    - [ ] 플레이어가 카드를 뽑을 경우 갱신된 카드 덱을 출력한다.
  - [ ] 딜러가 카드를 뽑았는지의 여부를 출력한다.
  - [ ] 최종덱과 총 점수를 출력한다.
  - [ ] 승패 여부와 결과를 출력한다.
