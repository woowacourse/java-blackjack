# java-blackjack

# 입력

✅ 게임 참가자 입력 뷰

```text  
게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)  
pobi,jason  
```  

---  

✅ 카드 수령 여부 입력 뷰

```text  
pobi는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)y  
```  

---

✅ 베팅 금액 입력 뷰

```text
pobi의 배팅 금액은?
10000
```

```text  
딜러와 pobi, jason에게 2장을 나누었습니다.  
```  

---  

✅ 게임 참가자, 딜러 카드 분배 결과 뷰

---


---  

✅ 딜러 카드 수령 여부 출력 뷰

```text  
딜러는 16이하라 한장의 카드를 더 받았습니다.  
```  

---  

✅ 참가자 및 딜러 최종 결과 출력 뷰

# 핵심 기능

✅ 게임 참가자 입력 및 검증 기능

- 게임에 참여할 사람의 이름을 입력 받는다.
- 게임 참가자 이름의 좌우 공백은 제거한다.
- 사용자의 입력은 ,(쉼표)로 구분한다.

⚠️ 기능 예외 사항

- 게임 참가 인원은 2~8명이 아니라면 예외를 발생 시켜야 한다.
- 게임 참가자 이름은 공백을 포함한 2~5글자가 아니라면 예외를 발생 시켜야 한다.
- 입력값에 공백만 입력되면 예외를 발생 시켜야 한다.
- 게임 참가자의 이름은 중복 되어선 안된다.

---  

✅ 카드 뽑기 기능

- 카드를 한 장 뽑는 기능입니다.
- 카드는 총 6벌이다.
- 중복될 수 있는 카드는 6개이다.
- 각 카드마다 6장을 넘어서면, 더 이상 출현되어선 안된다.

---

✅ 카드 분배 기능

- 카드는 총 6벌이다.
- 딜러, 게임 참가자들에게 카드를 2장씩 나누어준다.

---

✅ 카드 점수 계산 기능

- 카드의 점수를 계산하는 기능입니다.

---  

✅ 게임 참가자 카드 수령 여부 입력 및 검증 기능

- 입력값은 "y" 또는 "n" 뿐이다.

⚠️ 기능 예외 사항

- 공백만 들어가면 예외를 발생 시켜야 한다.
- 입력값에 "y", "n"을 제외한 다른 값이 들어오면 예외를 발생 시켜야 한다.

---  

☑️ 게임 참가자 추가 카드 뽑기 기능

- 참가자의 입력값이 "y"라면 카드를 뽑은 뒤, 참가자의 전체 카드를 출력한다.
- 입력값이 "n"이라면 카드를 뽑지 않고 참가자의 전체 카드를 출력한다.

---

✅ 딜러 카드 추가 수령 여부 판단 및 카드 뽑기 기능

- 딜러는 처음에 받은 2장의 합계가 16이하이면 반드시 1장의 카드를 추가로 받고, 17점 이상이면 추가로 받지 않는다.

---  

✅ 게임 결과 분석 기능

- 딜러와 모든 플레이어의 점수를 계산한다.

---

✅ 최종 승패 계산 기능

- 게임 참가자의 승패 여부를 계산한다.
- 딜러가 버스트인 경우에는 아래의 규칙을 따른다.
    - 버스트가 아닌 모든 참가자는 승이다.
    - 버스트인 모든 참가자는 패다.
- 딜러가 버스트가 아닌 경우에는 아래의 규칙을 따른다.
    - 딜러의 점수보다 낮거나 버스트인 참가자들은 패다.
    - 딜러의 점수와 같은 참가자들은 무다.
    - 딜러의 점수보다 높음 참가자들은 승이다.

---

✅ 베팅 결과 계산 기능

- 처음 두 장의 카드 합이 21일 경우 블랙잭이 되면 베팅 금액의 1.5배를 딜러에게 받는다.
- 딜러와 플레이어가 모두 동시에 블랙잭인 경우 플레이어는 베팅한 금액을 돌려받는다.
- 카드를 뽑으며 21을 초과할 경우, 베팅 금액 모두를 잃게 된다.
- 딜러가 21을 초과하면 그 시점까지 남아있던 플레이어들은 가지고 있는 패에 상관 없이 승리해 베팅 금액을 받는다.
