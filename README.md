# java-blackjack

블랙잭 미션 저장소

## 기능 명세
1. 게임에 참여할 사람의 이름을 입력한다.
- [x] 사용자 이름 입력 안내메시지를 출력한다.
- [x] 사용자 이름은 쉼표(,)를 기준으로 분리한다.
- [x] 사용자 이름을 올바르게 입력할 때 까지 입력을 받는다.
  - [x] 사용자는 1명 이상 25명 이하로 입력한다.
  - [x] 사용자 이름은 공백일 수 없다.

2. 사용자별 배팅 금액을 입력한다.
- [ ] 배팅 금액은 0보다 큰 정수만 가능합니다.
- [ ] 올바른 배팅 금액을 입력할 때 까지 다시 입력받는다.
  - [ ] 배팅 금액은 0보다 큰 정수이다.

3. 카드 배분 결과를 출력한다.
- [x] 딜러 -> 사용자 순으로 카드 배분 안내메시지를 출력한다.
- [x] 딜러와 사용자 모두 랜덤하게 카드를 배분한다.
    - [x] 딜러는 한 장, 사용자는 두 장의 보유한 카드를 출력한다.

3. 사용자들에게 카드 추가 여부를 물어본다.
- [x] 해당 사용자의 카드의 합이 21 초과인 경우 카드 추가 여부를 묻지 않는다.
  - [x] 카드의 숫자 계산은 카드 숫자를 기본으로 하며, 예외로 Ace는 1 또는 11로 계산할 수 있으며, King, Queen, Jack은 각각 10으로 계산한다.
- [x] 사용자 카드 추가 여부 입력 안내메시지를 출력한다.
- [x] 사용자 입력 (y/n)
    - [x] 사용자 입력이 'y'일 시, 하나의 랜덤 카드를 추가한다.
        - [x] 해당 사용자의 카드를 모두 출력한다.
    - [x] 사용자 입력이 'n'일 시, 해당 사용자의 카드가 두 개 이면 카드를 모두 출력한다.

4. 딜러의 카드 조건에 따라 카드 추가 여부를 출력한다.
- [x] 딜러 카드의 합이 16 이하이면, 하나의 랜덤 카드를 추가한다.
  - [x] 카드가 추가될 때 마다 안내 메시지를 출력한다.

5. 딜러 -> 사용자 순으로 카드 결과 및 합산 결과를 출력한다.
- [x] 각 보유한 카드 및 합산 결과를 출력한다.

6. 딜러 -> 사용자 순으로 최종 수익을 출력한다.
- [x] 딜러의 최종 수익을 출력한다.
- [x] 사용자의 수익을 출력한다.
- [x] 21을 초과하지 않으면서 21에 가까운 사람이 이긴다.
- [x] 딜러와, 사용자 모두 21을 초과한 경우 무승부 처리가 되며, 수익은 0원이다.
