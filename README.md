# java-blackjack

블랙잭 미션 저장소

## 프로그램 설명

Java로 구현한 콘솔 기반 블랙잭 게임입니다.

### 게임 흐름

1. **플레이어 이름 입력** - 쉼표(`,`)로 구분하여 여러 명의 플레이어 이름을 입력합니다.
2. **초기 카드 배분** - 딜러와 모든 플레이어에게 각 2장의 카드를 나눠줍니다. 딜러는 첫 번째 카드 1장만 공개합니다.
3. **플레이어 턴** - 각 플레이어는 카드를 추가로 뽑을지 여부를 `y/n`으로 선택합니다. 21점을 초과하면 버스트(패배)됩니다.
4. **딜러 턴** - 딜러는 16점 이하이면 자동으로 카드를 추가로 뽑습니다.
5. **최종 결과 출력** - 모든 참가자의 최종 카드와 점수, 승패 결과를 출력합니다.

### 주요 규칙

- **버스트**: 카드 합산 점수가 21을 초과하면 자동 패배
- **A(에이스)**: 카드 현재 점수에 따라 1 또는 11로 적용가능
- **J, Q, K**: 10점으로 계산
- **딜러**: 16점 이하일 경우 자동으로 카드를 추가 드로우
- **플레이어 이름**: 중복 불가, `딜러`라는 이름 사용 불가

### 기능 구현 목록

- 딜러
  - [x] 처음 카드를 두 장 받는다.
  - [x] 카드를 한장만 공개한다. (제약 : 처음만 공개)
  - [x] 두 카드의 합이 16이하일때 카드를 한장 더 받는다.

- 플레이어
  - [x] 처음에 카드를 두 장 받는다.
  - [x] 카드를 두장 전부 공개한다.
  - [x] 카드의 합이 21을 넘지 않을 경우, 카드를 뽑을 수 있다.
  - [x] 플레이어의 이름은 겹치면 안된다.
  - [x] 플레이어는 한명 이상이어야 한다.
  - [x] 플레이어는 딜러라는 이름을 쓰면 안된다.
  
- 랜덤 카드 생성
  - [x] 카드는 2~9,J,Q,K,A의 넘버링을 가진다.
  - [x] 카드는 하트, 다이아몬드, 스페이드, 클로버의 문양을 가진다.
  - [x] 카드는 중복되면 안된다.

- 점수 계산
  - [x] 카드는 숫자 넘버링일 경우 그 숫자로, J,K,Q의 넘버링일때는 10점의 점수를 가진다.
  - [x] A는 카드에 따라 1 또는 11의 점수를 가질 수 있다.
    - [x] A가 없을 경우, 카드의 합이 10이하일 경우에는 11점, 11점 이상일때는 1점을 가진다.
    - [x] A가 이전에 있을경우 그 A는 무조건 1점을 가진다.
  - [x] 플레이어의 점수는 카드의 합산 점수로 결정된다.

- 승자, 버스트 판정
  - [x] 21점을 넘으면 버스트로 판정되고 게임은 패배한다.
  - [x] 그렇지 않을 경우, 딜러보다 점수가 높은 사람은 승리한다.

- 입력
  - [x] 플레이어 이름을 입력한다.
    - [x] 이름은 쉼표 단위로 구분한다.
  - [x] 플레이어가 카드를 뽑고 싶을 때 y,n만을 입력한다.
    - [x] 다른 문자가 들어왔을 때 예외를 뱉는다.

- 출력
  - [x] 딜러의 첫번째 카드를 출력한다.
  - [x] 플레이어의 카드 덱을 출력한다.
    - [x] 플레이어가 카드를 뽑을 경우 갱신된 카드 덱을 출력한다.
  - [x] 딜러가 카드를 뽑았는지의 여부를 출력한다.
  - [x] 최종덱과 총 점수를 출력한다.
  - [x] 승패 여부와 결과를 출력한다.
