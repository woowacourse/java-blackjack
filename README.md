# java-blackjack

블랙잭 미션 저장소

### 기능 요구사항 체크리스트

#### step 1
- [x] 게임에 참여할 사람의 이름을 쉼표(,) 기준으로 입력받아 분리한다.
- [x] 0부터 51 사이의 난수를 생성해 무늬와 숫자가 매핑된 중복 없는 카드를 발급한다.
- [x] 게임 시작 시 딜러와 플레이어에게 각각 2장씩 카드를 분배한다.
- [x] 딜러의 초기 카드는 1장만 보여주고, 플레이어는 보유한 모든 카드를 보여준다.
- [x] 플레이어는 카드 합계가 21을 넘지 않는 한 계속해서 카드를 더 받을 수 있다.
- [x] 카드의 합을 계산할 때 K, Q, J는 10으로 계산하고, A는 11로 계산하고 21을 초과하면 1로 계산한다.
- [x] 딜러는 처음에 받은 2장의 합계가 16 이하이면 무조건 1장의 카드를 추가로 받고, 17 이상이면 받지 않는다.
- [x] 딜러와 플레이어의 턴이 모두 끝나면 각자의 최종 카드 목록과 점수를 출력한다.
- [x] 최종 점수를 비교해 딜러와 각 플레이어의 승/패/무 결과를 계산하고 출력한다.

#### step 2
- [ ] 딜러와 플레이어는 배팅금을 가진다.
- [ ] 딜러와 플레이어는 현재까지의 최종 수익을 가진다.
- [ ] 배팅금은 게임 시작 시, 이름과 함께 입력된다.
- [ ] 플레이어가 블랙잭으로 승리하면 배팅금이 1.5배가 된다.
- [ ] 플레이어가 승리하면 배팅금을 유지한다.
- [ ] 플레이어와 딜러가 모두 블랙잭이거나, 합산 점수가 같으면 배팅금을 돌려받는다. (배팅금이 0이 된다.)
- [ ] 플레이어가 패배하면 배팅금을 음수로 변환한다.
- [ ] 게임이 끝나면, 모든 플레이어의 현재 배팅금을 확인하여 플레이어의 수익에 덧셈, 딜러의 수익에 뺄셈 연산을 한다.

### 주요 로직 요약

#### step 1
- **카드 생성**
    - `Card`에서 `0~51` 범위의 난수를 뽑아 카드 번호를 만든다.
- **카드 이름/점수 매핑**
    - 카드 번호를 `shape(무늬)`와 `number(랭크)`로 분리해 문자열 카드명으로 변환한다.
    - 점수는 `A=11`, `2~10=숫자값`, `J/Q/K=10`으로 계산한다.
- **중복 없는 카드 보장**
    - `DuplicationSet`에 이미 나온 번호를 저장해, 중복 번호가 나오면 다시 뽑는다.
- **버스트와 A 점수 보정**
    - 플레이어 점수가 21을 초과하면, 보유한 A 개수만큼 `11 -> 1` 보정(10점 차감)을 적용한다.
- **게임 진행**
    - 시작 시 딜러/플레이어 모두 2장 배분
    - 플레이어는 입력(`y/n`)으로 추가 카드 선택
    - 딜러는 합계가 16 이하일 때 자동으로 추가 카드 드로우
- **승패 계산**
    - 버스트는 `-1` 처리 후 딜러와 각 플레이어 점수를 비교해 `승/패/무`를 기록한다.
    
#### step 2
- 배팅금 입력
  - 플레이어를 순회하며 배팅금을 입력받는다.
  - 배팅금은 플레이어 객체가 기억해야 하는 값이므로 필드로 존재해야한다.
- 수익금 초기화
  - 플레이어와 딜러는 수익금 필드를 가진다.
  - 수익금은 한 게임이 종료될 때, 배팅금을 더하거나 뺀다. (플레이어는 더하고, 딜러는 뺀다)
- 배팅금 갱신
  - 플레이어 승리 시 플레이어의 배팅금을 유지한다.
  - 무승부 시 플레이어의 배팅금을 0으로 갱신한다.
  - 플레이어 패배 시 플레이어의 배팅금을 음수로 만든다.
- 최종 수익 출력
  - 한 게임이 끝나면 최종 수익을 출력한다.

### 승패 계산
  - 플레이어가 BLACKJACK으로 승리하는 시나리오
    - 게임 시작 시에 플레이어의 카드 합이 21이고, 딜러는 21이 아닌 경우
  - 플레이어가 승리하는 경우
    - 플레이어는 STAND 상태이고, 딜러는 BUST인 경우
    - 플레이어와 딜러 모두 STAND인 상태에서 플레이어의 카드 합이 더 높은 경우
  - 무승부인 경우
    - 게임 시작 시 카드 2장으로 플레이어와 딜러 모두 BUST인 경우
    - 플레이어와 딜러 모두 STAND인 상태에서 플레이어와 딜러의 합이 같은 경우
  - 플레이어가 패배하는 경우
    - 플레이어가 HIT한 후에 BUST되는 경우
    - 플레이어와 딜러 모두 STAND인 상태에서 플레이어의 합이 딜러의 합보다 적은 경우
    - 플레이어는 STAND인 상황에서 딜러가 BLACKJACK인 경우
