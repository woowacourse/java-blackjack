# 🃏 java-blackjack

우아한테크코스 블랙잭 미션 저장소입니다.  
Java 기반으로 **객체지향 설계를 적용한 블랙잭 게임**을 구현합니다.

---

# 구현 기능

## 게임 규칙

- 플레이어와 딜러에게 **무작위 카드 지급**
- 처음 **2장씩 카드 분배**
- 플레이어 카드 전체 + 딜러 카드 1장 공개
- **Ace는 1 또는 11 중 최적의 값 선택**
- 플레이어는 **21 초과 전까지 Hit 가능**
- 딜러는 **16 이하일 경우 반드시 Hit**
- **21 초과 시 Bust → 패배**
- 게임 종료 후 **점수 및 승패 결과 출력**

# 객체 책임

### Card 도메인

**Card**  
카드 한 장을 표현하는 객체  
- `Rank`, `Suit` 정보를 보유  
- Ace 여부를 판단

**Rank**  
카드 숫자와 기본 점수 정의  
- 숫자 카드(2~10) → 해당 숫자  
- 문자 카드(J, Q, K) → 10점  
- 출력용 기호 관리

**Suit**  
카드의 모양(♠, ♥, ♦, ♣) 정의

**ShuffledDeck**  
게임에서 사용할 랜덤 카드 묶음  
- 객체 생성 시 랜덤 카드 초기화 
- 카드를 한 장씩 제공 (`draw`)

**Hand**  
참가자가 보유한 카드 집합
- 카드 추가
- 블랙잭 점수 계산
- Ace를 1 또는 11로 계산
- 최적 점수 선택
- Bust 여부 판단

---

### Participant 도메인

**Participant**  
플레이어와 딜러의 공통 상위 객체  
- 이름(`Name`)과 손패(`Hand`) 보유  
- 카드 추가(Hit) 기능 제공

**Player**  
게임에 참여하는 플레이어  
- Bust(21 초과)가 아닐 경우 카드 추가 가능

**Dealer**  
게임 규칙을 수행하는 딜러  
- 점수 16 이하 → 반드시 Hit

**Name**  
참가자 이름을 관리하는 값 객체  
- null / 공백 검증  
- 문자열 정제

**Bet**  
플레이어의 배팅 금액을 표현하는 값 객체
- 배팅 금액을 관리
- 게임 결과에 따라 수익을 계산

**PlayerGroup**  
플레이어 목록을 관리하는 일급 컬렉션
- 플레이어 목록 관리
- 플레이어 전체 딜링

**PlayerNames**  
플레이어 이름 목록을 관리하는 일급 컬렉션
- 플레이어 이름 목록 관리
- 이름 중복 검증

---

### Game 흐름

**BlackjackGame**
블랙잭 게임의 핵심 도메인 상태와 동작을 관리
- 덱, 딜러, 플레이어 그룹 관리
- 초기 카드 분배 수행
- 참가자에게 카드 hit
- 딜러 턴 진행
- GameReferee 을 통한 승패 판정

**Score**  
블랙잭 점수 비교
- 카드 총 합 관리

**GameResult**  
게임 결과와 배당률 정의
- 블랙잭 / 승 / 패 / 무

**BlackjackGameReferee**
- 블랙잭 규칙에 따라 점수를 비교하고 승패를 판정

---

### Game 흐름

**BlackjackApplication**
애플리케이션의 전체 실행 흐름을 조율
- 게임 생성
- 초기 카드 분배
- 각 플레이어의 Hit 여부 입력 및 턴 진행
- 딜러 턴 진행
- 점수 계산
- 게임 결과 출력

**BlackjackGameBootstrap**
애플리케이션의 생성에 필요한 설정 및 생성 책임
- 플레이어(이름, 배팅금액) 초기화
- BlackjackGame 생성

---

### View / DTO

**InputView**  
사용자 입력 처리

**OutputView**  
게임 진행 및 결과 출력

**DTO**  
도메인 데이터를 View로 전달하기 위한 객체
