# 블랙잭 Blackjack

## 요구사항

---

- [x] 게임에 참여할 사람의 이름 입력
  - [x] 참여자 이름 입력 프롬프트를 출력
    - [메시지] `게임에 참여할 사람의 이름을 입력하세요.(쉼표 기준으로 분리)`
  - [x] 참여자 이름 입력
    - [x] [예외]`IllegalArgumentException`: 이름이 서로 중복되는 경우
    - [x] [예외]`IllegalArgumentException`: 공백으로 시작하거나 끝나는 경우
- [ ] 플레이어들의 베팅 금액 입력
  - [ ] [출력] `xx의 베팅 금액은?`
  - [ ] [예외] 베팅 금액이 0원 또는 음수인 경우
- [x] 딜러와 참여자에게 카드 분배 및 출력
  - [x] 딜러와 참여자에게 카드 2장씩 분배
  - [x] 카드 분배 완료 메시지 출력
    - [형식] "딜러와 " + 참여자 이름들 + "에게 2장을 나누었습니다."
  - [x] 딜러 카드 출력
    - [형식] "딜러카드: " + 카드명
  - [x] 참여자들의 카드 출력
    - 형식: 이름 + "카드: " + 카드명1 + ", " 카드명2
- [x] 각 참여자 별로 카드 추가 분배
  - [x] 카드 추가 분배 여부 입력 프롬프트 출력
    - [형식] 이름 + `는 한장의 카드를 더 받겠습니까?(예는 y, 아니오는 n)`
  - [x] 카드 추가 분배 여부 입력
    - [x] [예외] `IllegalArgumentException`: 'y" 나 'n'이 아닌 경우(대문자 Y, N 불가능)
  - [x] 'y'가 입력되면 위의 과정 반복
- [x] 딜러의 카드 추가 분배
  - [x] 딜러의 핸드 점수 합이 16점 이하인지 확인
  - [x] 16점 이하이면 핸드에 카드 한 장 더 추가
    - [x] [메시지] `딜러는 16이하라 한장의 카드를 더 받았습니다.`
  - [x] 16점 초과할 때까지 반복
    - [x] [메시지] `딜러는 16초과라 더 이상 카드를 받지 않습니다.`
- [x] 최종 핸드 및 점수 계산/출력
  - [x] 딜러와 각 플레이어의 핸드 합 계산
    - [x] 2~10은 그대로, J, Q, K는 10점으로 계산 
    - [x] Ace는 1점 또는 11점 중 유리한 패로 계산
  - [x] 딜러와 각 플레이어의 핸드와 총 점수 출력
    - [형식] `딜러카드: 3다이아몬드, 9클로버, 8다이아몬드 - 결과: 20`
    - [형식] `[플레이어]카드: 2하트, 8스페이드, A클로버 - 결과: 21`
- [ ] 최종 승패 계산 및 최종 수익 출력
  - [x] 각 플레이어와 딜러의 점수 비교
  - [ ] 딜러와 플레이어 최종 수익 계산
    - [ ] 플레이어는 이기면 베팅 금액만큼 얻고, 비기면 그대로, 지면 베팅 금액만큼 잃는다.
    - [ ] 플레이어의 첫 핸드가 블랙잭(21)이면 베팅 금액의 1.5배로 받는다.
    - [ ] 딜러의 최종 수익은 `(플레이어의 최종 수익 총합) x (-1)`이다.
  - [ ] 최종 수익 출력
    - [ ] 딜러 수익 출력
    - [ ] 각 플레이어 수익 출력
