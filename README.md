# java-blackjack

## 구현할 기능
- [x] 플레이어와 딜러에게 무작위 카드를 지급한다.
- [x] Ace 카드 값을 1 또는 11로 조정한다.
- [x] 카드 총 합계를 계산한다.
- [x] 플레이어의 버스트 여부를 판단한다.
- [x] 딜러가 카드를 더 뽑아야 하는지 판단한다.
- [x] 플레이어별 승패 여부를 판단한다.
- [ ] 플레이어는 게임을 시작할 때 배팅 금액을 정해야 한다.
- [ ] 카드를 추가로 뽑아 21을 초과할 경우 배팅 금액을 모두 잃게 된다.
- [ ] 처음 두 장의 카드 합이 21일 경우 블랙잭이 되면 베팅 금액의 1.5 배를 딜러에게 받는다.
- [ ] 딜러와 플레이어가 모두 동시에 블랙잭인 경우 플레이어는 베팅한 금액을 돌려받는다.

### 객체의 책임
#### InputView
> 콘솔에서 입력을 받아 모델에 넘긴다.

#### OutputView
> 모델에서 받은 데이터 등을 콘솔에 출력한다.

#### BlackjackGame
> 게임의 규칙과 순서를 관리한다,

#### Card
> rank 와 suit 데이터를 관리한다.

#### Hand
> 플레이어/딜러가 소유하는 여러개의 카드를 관리한다.

#### Deck
> 플레이어, 딜러에게 지급되는 여러개의 카드를 관리한다.

#### Participant
> Player, Dealer 클래스의 상위 클래스이다. 참가자의 기본적인 상태와 기능을 담당한다.

#### Player
> Participant 의 하위 클래스로, 플레이어의 상태와 기능을 관리한다.

#### Dealer
> Participant 의 하위 클래스로, 딜러의 상태와 기능을 관리한다.
