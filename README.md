# java-blackjack

블랙잭 미션 저장소

## 기능 요구 사항

1. 입력
- 플레이어(쉼표 기준으로 분리)

2. 랜덤 카드 생성
- 숫자, 문양별로 랜덤 카드 생성
  - 숫자, 문양별 enum 관리

3. 카드 드로우
- 딜러 2장, 플레이어별 2장 카드 드로우
  - 딜러는 1장만 출력

4. 카드 추가 드로우(기능 루프)
- 플레이어별 추가 드로우 선택(y/n)
  - 추가 드로우 시
    - 현재 패 출력
    - 현재 패 값 계산
- 딜러 추가 드로우
  - 딜러 패의 값이 16이하라면 한장씩 추가

5. 버스트 확인
- 카드 패 합산 결과 21 초과 시 버스트

6. 출력
- 딜러 드로우 종료 시, 최종 결과 출력
