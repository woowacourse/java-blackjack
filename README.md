# java-blackjack

블랙잭 미션 저장소

- [x] 카드 점수 계산 (숫자는 그대로, J,Q,K는 10으로 계산)
- [ ] 카드 합계 계산
    - [ ] Ace제외 카드 합계 계산
    - [ ] Ace 1/11 처리
    - [ ] 전체 합계 계산
- [ ] 버스트 판정 (합계 > 21 인지 판단)
- [ ] 카드 중복없이 랜덤 추출
- [ ] 카드 지급 (특정 사람에게 카드 1장 지급)
- [ ] 딜러 카드 합이 17이상이 될때까지 카드 지급
- [ ] 승패 결과 계산
    - 둘다 버스트가 아닌 경우
        - [ ] 둘 중 카드합이 큰 사람 승리
        - [ ] 카드합이 같으면 무승부
    - 한명이 버스트인 경우
        - [ ] 버스트인 사람 패배
    - 둘다 버스트인 경우
        - [ ] 무승부

- [ ] 게임진행
    - [ ] 초기 시작시 인당 카드 2장씩 지급
    - [ ] 참가자 한명이 버스트가 되거나 카드를 안받는다 선택한 경우 다음 참가자 차례로 넘어감
    - [ ] 마지막 참가자가 버스트가 되거나 카드를 안받을 경우 게임 종료

-[ ] 입력
    - [ ] 게임 참가자 이름 입력
    - [ ] 카드 더받기 유무 입력
- [ ] 출력
    - [ ] 분배 받은 카드 목록 출력
    - [ ] 딜러와 참가자 카드 합계 결과 출력
    - [ ] 최종 승패 결과 출력
